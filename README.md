# leetcode-go

my ([profile][profile]) solutions to Leetcode problems in Golang

## structure

```
.
├── src/                  - problem solutions & tests for them
│   ├── pX/                 - problem X
│   │   ├── p1_test.go        - test
│   │   └── p1.go             - solution
│  ...
└── utils/                - helpers for tests & common functions,
                            e.g., `ListNode` struct
```

## usage

to format ...

```sh
go fmt ./...
```

to test ...

```sh
go test ./...
```
